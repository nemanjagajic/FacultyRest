# FacultyRest

Simple rest client to manipulate faculties and students.

Request that can be made:
Students:
student /getAll
        /get/{id}
	      /add
	      /update{id}
	      /remove

Faculty:
faculty /getAll
	      /get/{id}
	      /add
	      /update{id}
	      /remove
	      /addStudents/{id}
	      /removeStudents/{id}
